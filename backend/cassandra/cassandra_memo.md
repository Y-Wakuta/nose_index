poklet/cassandra:latestはベースになっているlinuxが最小構成版でapt等も使えなかったので、spotifyのcassandraイメージを使用

### dockerを直に使う時のコマンド

```sh
docker run -it --rm --net container:cassandra -v `pwd`:/data nuvo/docker-cqlsh bash

# cqlsh cassandra
```
